## 🚀 Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jaewonparkk/leetcode.git
   cd leetcode
   ```

2. **Navigate to a difficulty folder:**
   ```bash
   cd easy  # or medium, or hard
   ```

3. **Run a solution:**
   ```bash
   python 1.two-sum.py
   ```
